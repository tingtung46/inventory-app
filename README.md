# Inventory App

An application built using nodejs as backend. The user can store an item's info into inventory.

## Structure of App

1. A page to display all items on table.
2. A page to add item's info.
3. A page to update/edit item's info.
4. A page to add and delete categories and brands.
5. A search page to display searched item.
6. Search bar to search an item.

## Feature of App

- Add and item's info then display it into table.
- Delete item from database.
- Insert some keyword into search bar to search item.
- Add and delete categories or brands.
- If some category of brand on sidebar are not available/had been deleted, users can go to "Other" to take a look the items that were not contained on available ones.
